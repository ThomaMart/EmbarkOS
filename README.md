# EmbarkOS

EmbarkOS est une distribution Linux embarquée basée sur Buildroot.

L'objectif du projet est de comprendre et maîtriser la création complète d'un système Linux embarqué :

- génération d'une image Linux
- cross-compilation
- configuration du noyau
- personnalisation du système
- création de packages Buildroot
- intégration de logiciels embarqués
- automatisation du processus de build

Le projet cible principalement les plateformes ARM, en particulier Raspberry Pi.
